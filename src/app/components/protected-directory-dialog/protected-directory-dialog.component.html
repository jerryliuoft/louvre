warning Protected Directory Selected

The browser's security settings prevent web applications from accessing certain restricted system folders, such as your Downloads, Desktop, or Documents folder directly.

If you encountered an error, or the folder appeared to be empty when it is not, it is likely due to this protection.

Workaround:

  1. Create a dedicated folder for Louvre elsewhere (e.g., inside Documents, or directly on your C:\ drive).
  2. Move your media into this new folder.
  3. Select the new folder using the picker.
